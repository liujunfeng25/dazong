# 工厂端（指定厂家）

入口：`/factory/orders`

## 1. 角色定位

工厂端是指定厂家商品的特殊供货方，只处理同时满足以下条件的分单行：

- 商品标记为指定厂家商品。
- 商品配置的指定厂家是当前账号。
- 配送商已将该商品行分配给当前厂家。

当前系统没有进厂登记、加工任务、生产排程或成品入库流程。

## 2. 查看订单

1. 在 `/factory/orders` 按订单状态、订单号、创建日期或配送日期筛选。
2. 打开 `/factory/orders/{id}`。
3. 页面只显示当前厂家负责的指定商品行。
4. 核对数量、价格、分单状态和质检要求。

## 3. 质检报告

- `/factory/reports/upload` 上传批次质检报告。
- `/factory/periodic-reports` 维护周期质检报告。
- `/factory/reports` 查看报告。

报告是否可用于当前订单取决于商品质检模式、报告状态和覆盖日期。

## 4. 出库

指定厂家出库逻辑与供货方相同：

1. 检查分单行。
2. 满足质检要求。
3. 打印标签。
4. 确认出库。
5. 等待配送中心 PDA 分检。

## 5. 账单

入口：`/factory/bills`

厂家查看对配送商的应收账单，金额只包含分配给本厂家的指定商品行。
