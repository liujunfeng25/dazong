# 供货方

入口：`/supplier/orders`

## 1. 数据权限

供货方只看到配送商分配给自己的订单行、金额和质检要求。没有分配给本账号的商品不会出现在列表和详情中。

供货方不存在独立库存模块，也不存在旧版“配货任务”页面。

## 2. 商品报价

入口：`/supplier/quotes`

1. 按商品名或编码搜索普通商品。
2. 查看参考价、合约浮动价和当前报价。
3. 填写大于零的供货报价。
4. 批量保存。

指定厂家商品不进入普通供货方报价范围。没有有效配送合约时，页面会提示无法计算合约浮动价。

## 3. 订单履约

1. 在 `/supplier/orders` 查看待发货订单。
2. 打开 `/supplier/orders/{id}`。
3. 核对只属于本供货方的分单行。
4. 按质检模式提交或关联报告。
5. 打印分单标签。
6. 确认实物和标签一致后执行出库。

每条分单出库后状态为“已出库”。订单是否进入后续阶段由全部分单的出库和 PDA 分检进度共同决定。

## 4. 质检

- 批次质检：针对订单或分单提交报告。
- 周期质检：针对商品和供货方维护覆盖日期的报告。

若报告缺失、待审核或已退回，页面会提示补充或修订。

## 5. 账单

入口：`/supplier/bills`

供货方通常查看对配送商的应收账单。金额以分配给本供货方的订单行为准，不以整单总金额为准。
