# 司机端

项目：`/Users/Admin/Project/driver-android`

## 1. 登录

1. 输入 API 根地址。
2. 输入启用车辆的车牌号。
3. 输入司机密码。
4. 系统签发独立 driver token。

车牌必须在配送商车辆管理中存在且启用。重复车牌会阻止登录。

## 2. 今日车次

司机只能查看分配给当前车辆、计划日期为当日且状态属于待发车、有阻塞、运输中或已完成的车次。

车次卡展示：

- 路线号、车辆和司机。
- 站点数量。
- 可装/已装和阻塞数量。
- 计划出发时间和状态。

## 3. 开始配送

司机端不能代替调度员发车。只有调度端已将车次设为“运输中”后，司机才能开始配送。

## 4. 地图与导航

- 车次详情使用高德 Web 地图展示仓库、站点、路线和车辆位置。
- 手机位置可作为车辆位置缺失时的展示回退。
- 导航按钮可外跳高德地图 App。
- VPN、网络或 Key 异常时地图会显示占位，但站点清单和确认送达仍可使用。

## 5. 确认送达

1. 到达当前站点。
2. 核对食堂、地址和商品。
3. 点击确认送达。
4. 订单从“发货”进入“收货”。
5. 采购方或智能秤继续完成收货确认。

全部站点送达后车次自动完成。
