## 边界

- 六类 PC 业务角色和三个 Android 专用终端构成九端协作范围。
- 工厂端是指定厂家履约端，不是加工生产管理系统。
- 操作手册只写当前代码可执行的功能。
- 完整接口以 FastAPI `/docs` 和前端 `/docs` 为准。
- 本目录是监管 AI “怎么操作”问题的唯一文档来源。
