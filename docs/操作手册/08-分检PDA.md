# 分检 PDA

项目：`/Users/Admin/Project/sorter-pda-android`

## 1. 登录

使用配送商账号登录。PDA 只允许查看当前配送商的分单。

## 2. 今日分检

首页显示选定配送日期的：

- 分单总数。
- 已分检。
- 待分检。
- 未出库/异常。
- 完成百分比。

默认使用当日日期，也可以切换查询日期。

## 3. 扫码

1. 确认设备编码。
2. 扫描供货方标签。
3. 查看成功、重复或错误反馈。
4. 错误时按页面原因处理，不连续盲扫。

支持 Urovo 常见扫码广播和手工输入。

兼容的条码包括：

```text
DZALLOC:<分单ID>
DZALLOC-<分单ID>
ALLOC:<分单ID>
ALLOC-<分单ID>
裸分单ID
```

## 4. 错误含义

- 条码不存在：不是系统生成的有效分单标签。
- 不属于当前配送商：货品进入了错误分检场。
- 订单已取消：停止分检并交现场主管。
- 非当日订单：核对标签日期和调度计划。
- 供货商尚未出库：先完成供货出库。
- 已分检：系统已存在扫码记录，不重复计数。

## 5. 网络检查

设置页可调用健康检查。健康检查成功只代表 API 可达，不代表当前账号、订单或条码一定有效。
